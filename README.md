# VPS Automation Toolkit
### by varadbamalwa

This is a **VPS Automation Tool** that helps you quickly set up:

- XRDP (Remote Desktop)
- Tailscale VPN
- Pterodactyl Panel
- Pterodactyl Wings
- KVM Host setup
- Create Real KVM VPS

---

## Features

- Easy **menu-based installer**
- ASCII banner logo for branding
- Works on Debian/Ubuntu
- Fully **open-source** on GitHub
- Scripts organized in `/scripts` folder

---

## Installation & Usage

Clone this repository:

```bash
git clone https://github.com/VaradPlayzYt/vps-automation.git
cd vps-automation
chmod +x install.sh scripts/*.sh
./install.sh
